A simple, Netflix-inspired landing page that resembles the appearance and feel of Netflix's homepage using only HTML and CSS.
The layout structure, visual consistency, innovative UI styling, and responsive design principles are the main topics of this project.

Features

 -A fully designed UI that resembles Netflix
 -Layout that adapts to various screen sizes
 -The "Trending Now" section can be scrolled horizontally
 -A background overlay for the hero section
 -UI elements that are structured, such as the footer, sections, and navigation bar
 -Lightweight, quick, and suitable for beginners (HTML + CSS only) project

Tech Stack :
 
Technology         Purpose
 
HTML5              Page Structure
CSS3               Responsiveness, Style, and Layout
